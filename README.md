# Машина Тьюринга эмулятор (C++/Qt)
### Основные функции:
* **Важные кнопки:** Автоматический (`Play`), пошаговый (`Step`), пауза и сброс.
* **Скорость:** Регулировка задержки шага от *100* до *2000* мс.
### Инструкция:
* **Алфавит:** Введите символы и нажмите кнопку **Задать алфавит**.
* **Программа:** В ячейках таблицы укажите правила в формате: `символ, направление, состояние`.
	* *Пример:* `1, R, q1` (записать 1, сдвиг вправо, переход в q1).
    * *Пример:* `λ, L, !` (записать лямбду, сдвиг влево, остановка).
* **Лента:** Введите слово в поле *Input String* -> нажмите **Задать строку**.
